# README

### Hi there! This is a personal project I have been working on. I hope you like it!

### Get this puppy up and running
1. fork, clone, bundle
1. rake db:create db:migrate
1. rails s
1. register and play!

### Live app
* [ten-000.herokuapp.com/](http://ten-000.herokuapp.com/)
